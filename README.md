TraceChain is a decentralized supply chain tracking application built on the Polygon blockchain, designed to bring transparency and trust to product journeys from manufacturer to end consumer. It uses Solidity smart contracts with role-based access control to record every ownership transfer and status update immutably on-chain, while storing rich product metadata — images, batch details, and specifications — on IPFS via Pinata to keep gas costs minimal. The React frontend connects to users' MetaMask wallets through Ethers.js, allowing manufacturers, distributors, and retailers to create products, update their status in real time, and transfer custody across the supply chain, with every action permanently verifiable on Polygonscan. Built as a full-stack Web3 portfolio project using Hardhat for smart contract development and testing, Polygon Mumbai testnet for free deployment, and Vercel for frontend hosting, TraceChain demonstrates how blockchain technology can eliminate counterfeit goods, reduce fraud, and give consumers a scannable, tamper-proof history of any product they hold.
